Language Quiz

A flashcard application for learning new words quickly and efficiently.

This application is a Flask-based backend that handles various functions related to speech recognition, text-to-speech conversion, and managing audio and JSON files.

Learn more

Features

  • Sentences Learning
  • Multi Learning
  • Scattered Words Learning
  • Add Lists of Words
  • Add New Word
  • Single Word Learning
  • Read from File
  • Edit List of Words

Backend Documentation

Installation and Configuration

Clone the repository and install dependencies:

git clone <REPOSITORY_URL>
cd <REPOSITORY_NAME>
pip install -r requirements.txt

Set environment variables:

import os
os.environ["CUDA_VISIBLE_DEVICES"] = ""
os.environ["SUNO_USE_SMALL_MODELS"] = "1"

Run the application:

python app.py

API Endpoints

The following endpoints are available:

  • Real-Time Speech Recognition: POST /real-time-speech-recognition
  • Send Static File: GET /<path:path>
  • Send Audio File: GET /audio_files/<path:path>
  • Text-to-Speech: POST /text-to-speech
  • Load Audio Paths: POST /load-audio-paths
  • Save JSON: POST /save
  • Save Statistic: POST /save_statistic
  • Save Setting: POST /saveSetting
  • Get Words: GET /words
  • Edit Word: POST /edit/<file>/<index>
  • Get Files: GET /files
  • Upload File: POST /upload

Directory Structure

  • uploads/: Stores uploaded files.
  • image_files/: Stores image files.
  • audio_files/: Stores audio files.
  • logs/: Stores application logs.
  • statistic/statistics.json: Stores saved statistics.
  • setting/excludedWords.json: Stores settings.
  • templates/: HTML templates (Jinja2).
  • static/: Static files (JS and CSS).
